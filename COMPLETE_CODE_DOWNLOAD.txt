## STEP 1: CREATE PROJECT FOLDER
------------------------------------
1. Create a folder called "realtyvibe" on your computer
2. Open terminal/command prompt in that folder


## STEP 2: INITIALIZE PROJECT
------------------------------------
Run these commands in your terminal:

npm init -y
npm install react@19.0.0 react-dom@19.0.0 react-router@7.1.1
npm install @react-router/node@7.1.1 @react-router/serve@7.1.1 isbot@5.1.0
npm install @radix-ui/react-accordion@1.2.2 @radix-ui/react-alert-dialog@1.1.4
npm install @radix-ui/react-aspect-ratio@1.1.1 @radix-ui/react-avatar@1.1.2
npm install @radix-ui/react-checkbox@1.1.3 @radix-ui/react-collapsible@1.1.2
npm install @radix-ui/react-context-menu@2.2.4 @radix-ui/react-dialog@1.1.4
npm install @radix-ui/react-dropdown-menu@2.1.4 @radix-ui/react-hover-card@1.1.4
npm install @radix-ui/react-label@2.1.1 @radix-ui/react-menubar@1.1.4
npm install @radix-ui/react-navigation-menu@1.2.3 @radix-ui/react-popover@1.1.4
npm install @radix-ui/react-progress@1.1.1 @radix-ui/react-radio-group@1.2.2
npm install @radix-ui/react-scroll-area@1.2.1 @radix-ui/react-select@2.1.4
npm install @radix-ui/react-separator@1.1.1 @radix-ui/react-slider@1.2.1
